  2. 통계에 기반한 tag 문법

    3. 통계에 따라 태그 문법을 학습해야한다!

    <p>: 문단 나누기인데 여백이 정해져있다
    <br>: 줄바꿈이고 CSS를 통해 여백 간격 조정 가능함
           : 닫히는 태그 존재 X
    <li>: 목차를 나타내는 태그 (자식태그)
    <ol>: <li>태그와 같은 항목들의 경계를 그룹해주는 tag(부모태그/ordered list)
           : 숫자로 list 표현
    <ul>: <li>태그와 같은 항목들의 경계를 그룹해주는 tag(부모태그/unordered list)
           : 점으로 list 표현
    <title>문법: 웹페이지의 제목을 설정하는 문법으로 검색엔진에 걸리도록하는 역할
    <meta charset>: 웹페이지가 숫자를 인식하도록 하여 문자를 표현해줌
    <a>: 하이퍼텍스트를 연결하는 태그 (ancheor:돛)
    - CSS: 속성(attribute)을 나타내는 것 -
    Ex. href: 링크 연결
    Ex. _blank: 연결된 링크 새탭으로 열기
